║                         🚀 CARA BUILD & RUN                                  ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. Copy .env file:
   ─────────────────────────────────────────────────────────────────────────
   cd C:\Users\Bisma\Desktop\PS_Demo\supoclip-dev\supoclip
   cp .env.example .env


2. Edit .env (WAJIB isi API key):
   ─────────────────────────────────────────────────────────────────────────
   notepad .env

   # Minimal config:
   GOOGLE_API_KEY=your_key_here
   TRANSCRIPTION_PROVIDER=google_genai
   ENABLE_SMART_CROPPING=true


3. Build dengan Docker:
   ─────────────────────────────────────────────────────────────────────────
   docker-compose build

   Waktu build: ~3-5 menit (first time)
   Dependencies yang akan di-install otomatis:
   - ultralytics (YOLO)
   - opencv-python
   - moviepy
   - numpy
   - 30+ dependencies lainnya


4. Run application:
   ─────────────────────────────────────────────────────────────────────────
   docker-compose up

   Services:
   - Frontend: http://localhost:3000
   - Backend:  http://localhost:8000
   - Redis:    localhost:6379
   - Postgres: localhost:5432


5. Test smart cropping:
   ─────────────────────────────────────────────────────────────────────────
   - Upload video via http://localhost:3000
   - Check logs: docker-compose logs -f backend
   - Look for: "Using strategy: stacking for 2 people"
